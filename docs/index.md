## Quick Start

### Prerequisites

- Android device with **developer options** enabled
- **ADB tools** installed on your development machine
- For Shizuku apps: [Shizuku](https://github.com/RikkaApps/Shizuku) installed and running

### Getting Started

1. Browse the sections above to find the tools you need
2. Follow installation instructions for specific applications
3. Configure ADB settings using the [ADB Commands](ADB-Commands.md) guide
